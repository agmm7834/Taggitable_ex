# Django Taggit Demo

Небольшой демонстрационный проект Django, показывающий, как подключить пакет `django-taggit` и организовать работу с тегами для статей.

## Возможности
- создание статей с тегами через веб-форму;
- фильтрация ленты по тегам;
- просмотр всех материалов, связанных с конкретным тегом;
- регистрация модели в административной панели.

## Требования
- Python 3.12+
- Django 5.1

## Установка
```bash
git clone <repo-url>
cd Taggitable_ex
python -m venv .venv
.venv\Scripts\activate  # Windows
pip install -r requirements.txt
```

## Настройка базы
```bash
python manage.py migrate
```

## Создание суперпользователя (по желанию)
```bash
python manage.py createsuperuser
```

## Запуск сервера
```bash
python manage.py runserver
```

После запуска приложение будет доступно по адресу <http://127.0.0.1:8000/>.

## Как работать с тегами
- Перейдите по ссылке «Новая статья» и создайте материал, указав список тегов через запятую.
- На главной странице отображаются все статьи с активным фильтром по выбранному тегу.
- Страница `Тег` показывает публикации, связанные с конкретной меткой.

## Дополнительно
- В административной части (`/admin`) можно управлять статьями и тегами.
- Для изменения внешнего вида обновите файлы в `static/css/style.css`.
